## 📖 About The Project

The useReducer Hook is one of React's powerful state management tools. It provides an alternative approach to useState when managing complex state transitions.

This project was created to explain:

- How reducers work
- Why useReducer exists
- Managing multiple state values
- Dispatching actions
- Writing predictable state logic
- Organizing scalable React applications

Whether you're a beginner learning React Hooks or an experienced developer preparing for interviews, this project serves as a practical learning resource.

----

## 🚀 Why useReducer?

While useState works great for simple states, managing multiple related states can become difficult.

Example:

const [count, setCount] = useState(0);

const [loading, setLoading] = useState(false);

const [error, setError] = useState(null);

As applications grow, state logic becomes scattered.

useReducer solves this problem by centralizing state updates into a single reducer function.
